### **心理学经典书籍分类整理清单**

#### **1. 行为决策与认知偏差（理解大脑如何犯错）**

这一类书籍主要探讨人类思维的局限性、非理性行为以及社会影响力的运作机制。

| 书名                         | 作者             | 核心看点                                             |
| ---------------------------- | ---------------- | ---------------------------------------------------- |
| **《影响力》**         | 罗伯特·西奥迪尼 | 剖析互惠、承诺、社会认同等6大说服力法则。            |
| **《思考，快与慢》**   | 丹尼尔·卡尼曼   | 区分直觉系统与逻辑系统，揭示决策中的系统性偏见。     |
| **《怪诞行为学》**     | 丹·艾瑞里       | 实验证明人类在金钱、情绪等方面极其规律的非理性行为。 |
| **《清醒思考的艺术》** | 鲁尔夫·多贝里   | 52个思维误区清单，帮助识别生活中的逻辑陷阱。         |

#### **2. 自我成长与认知觉醒（驱动自我进化）**

侧重于如何通过心理学知识改变底层逻辑，克服天性，实现个人能力的增长。

| 书名                     | 作者                 | 核心看点                                           |
| ------------------------ | -------------------- | -------------------------------------------------- |
| **《认知觉醒》**   | 周岭                 | 结合脑科学，教你如何开启自我驱动力，告别焦虑。     |
| **《终身成长》**   | 卡罗尔·德韦克       | 提出“成长型思维”，强调相信努力比相信天赋更重要。 |
| **《心流》**       | 米哈里·契克森米哈伊 | 探讨如何进入极度专注的幸福状态，提升生活质量。     |
| **《自卑与超越》** | 阿尔弗雷德·阿德勒   | 探讨自卑感的补偿机制及如何通过社会兴趣超越自我。   |

#### **3. 情感关系与治愈系（处理人际与心理创伤）**

关注亲密关系、社交心理以及心理咨询过程中的自我疗愈。

| 书名                               | 作者             | 核心看点                                             |
| ---------------------------------- | ---------------- | ---------------------------------------------------- |
| **《亲密关系》**             | 罗兰·米勒       | 情感领域的“百科全书”，基于实证科学解析吸引与冲突。 |
| **《也许你该找人聊聊》**     | 洛莉·戈特利布   | 以治疗师和来访者的双重视角，探讨改变与失去。         |
| **《蛤蟆先生去看心理医生》** | 罗伯特·戴博德   | 童话外壳下的心理干货，深入浅出讲解TA交互疗法。       |
| **《被讨厌的勇气》**         | 岸见一郎         | 以对话形式阐述阿德勒心理学，教你获得心灵的自由。     |
| **《身体从未忘记》**         | 巴塞尔·范德考克 | 揭示心理创伤如何生理性地改变大脑，以及疗愈之道。     |
| **《非暴力沟通》**           | 马歇尔·卢森堡 | 提出观察-感受-需要-请求四步法，从根本上改善沟通与关系。 |

#### **4. 进化视角与社会观察（洞察人性底层逻辑）**

从生物进化和社会群体的角度，解释人类本能和社会现象。

| 书名                     | 作者           | 核心看点                                           |
| ------------------------ | -------------- | -------------------------------------------------- |
| **《道德动物》**   | 罗伯特·赖特   | 用进化心理学解释爱情、友谊、道德背后的生存逻辑。   |
| **《乌合之众》**   | 古斯塔夫·勒庞 | 探讨群体心理，解释个人在群体中为何会丧失理智。     |
| **《正义之心》**   | 乔纳森·海特   | 揭示道德直觉如何运作，以及为何好人会因政见而分裂。 |
| **《金钱心理学》** | 摩根·豪泽尔   | 探讨财富背后的贪婪、恐惧与幸福感，非理财工具书。   |
| **《社会性动物》** | 埃利奥特·阿伦森 | 社会心理学经典，解析从众、偏见、攻击等社会行为背后的心理机制。 |

#### **5. 系统入门必备（构建专业知识框架）**

如果你希望系统性地学习心理学，而不只是看碎片化的观点，以下是首选。

* **《心理学与生活》** （理查德·格里格）：全球最经典的心理学教材，涵盖感知、记忆、人格、障碍等所有分支，权威且易读。

你可以根据文档的用途（比如个人书单、读书笔记模板等）对上述表格进行微调。建议从《蛤蟆先生去看心理医生》或《影响力》开始读起，这两本的趣味性和实用性结合得最好。
